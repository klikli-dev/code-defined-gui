# Code-Defined GUI    

A minecraft mod that allows other mods to define GUIs in code - without the need to create the full layout in an image editor.
This allows for faster creation and prototpying of GUIs, and enables dynamic GUI layout changes without needing multiple hand-made background images to swap between.
