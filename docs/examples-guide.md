# Examples guide

CDG ships with a small `example.*` namespace for reference code.

## Current example

- `com.klikli_dev.codedefinedgui.example.ExampleLayouts`
- `com.klikli_dev.codedefinedgui.example.ExampleMenu`
- `com.klikli_dev.codedefinedgui.example.ExampleScreen`

## What examples are for

- understanding the intended integration flow
- copying small patterns into your own mod
- seeing how layout, menu binding, and screen resolution fit together

## What examples are not

- not stable API
- not a compatibility guarantee
- not the same thing as `internal.*` debug plumbing

If you want supported imports, use `api.*` and `premade.*`. Use `example.*` as reference only.
